## 📋 Описание

Реализация абстрактного типа данных (АТД) «Строка» на основе полиморфного динамического массива (`Vector`) с использованием концепции `FieldInfo` для управления типами данных.

Проект демонстрирует принципы объектно-ориентированного программирования на языке C:
- Полиморфизм через таблицу функций (`FieldInfo`)
- Инкапсуляция через непрозрачные типы
- Управление памятью с автоматическим освобождением ресурсов

## 🚀 Быстрый старт

### Требования
- Компилятор GCC (MinGW-w64)
- CMake 3.10 или выше
- Windows PowerShell / Linux Terminal / macOS Terminal

### Сборка проекта

```powershell
# 1. Перейти в папку проекта
cd C-LR-1

# 2. Настроить сборку
cmake -B build -G "MinGW Makefiles" -DCMAKE_BUILD_TYPE=Debug

# 3. Скомпилировать
cmake --build build

# 4.1. Запустить demo.exe

.\build\demo.exe

# 4.2. Или прогнать по тестам

.\build\tests\run_tests.exe
